# 🚀 Projet DevOps de Bout en Bout sur AWS

## Présentation

Ce dépôt documente mon parcours de montée en compétences vers le métier d'Ingénieur DevOps à travers la réalisation d'une plateforme Cloud moderne sur AWS.

L'objectif est de mettre en pratique les concepts DevOps les plus utilisés en entreprise : Infrastructure as Code, CI/CD, conteneurisation, Kubernetes, GitOps, supervision et sécurité.

---

## Objectifs du projet

* Construire une plateforme DevOps complète sur AWS
* Déployer l'infrastructure avec Terraform
* Mettre en place une chaîne CI/CD avec Jenkins
* Conteneuriser une application avec Docker
* Déployer sur Amazon EKS (Kubernetes)
* Implémenter le GitOps avec ArgoCD
* Superviser la plateforme avec Prometheus et Grafana
* Documenter l'ensemble de l'apprentissage et du projet
